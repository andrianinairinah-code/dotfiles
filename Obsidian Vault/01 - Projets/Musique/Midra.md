# Midra

- **GitHub**: https://github.com/XIAODUOLU/Midra
- **Licence**: Apache 2.0
- **Stack**: Python, agentic LLM
- **Local**: `D:\musique\MUSIC IA\Midra\`

Prompt-to-code MIDI composition framework. Transforme du langage naturel en code MIDI structuré, éditable et réutilisable.

## Utilité pour le projet
Générer des patterns d'exercices (gammes, arpèges, rythmes) à partir de prompts.
