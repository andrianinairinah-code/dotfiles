# ACE-Step

- **GitHub**: https://github.com/ace-step/ACE-Step
- **Stars**: ~4.5k
- **Licence**: Apache 2.0
- **Stack**: Python, PyTorch, Diffusion
- **Local**: `D:\musique\MUSIC IA\ACE-Step\`

Génération audio rapide (4min en 20s sur A100). Modèle foundation pour musique.
Support voix, lyric editing, remixing, track generation.

## Usage
```bash
pip install -e .
ace-step --prompt "blues progression in A" --output output.wav
```
