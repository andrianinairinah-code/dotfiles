Transforme n'importe quel codebase en graphe de connaissances interactif. Multi-agent pipeline (scanner, analyseur, architecte, guide, reviewer).

## Commandes
```
/understand              → analyser le codebase actuel
/understand-dashboard   → ouvrir le dashboard interactif
/understand-chat        → poser des questions sur le code
/understand-diff        → analyser l'impact des changements
/understand-explain     → expliquer un fichier/fonction
/understand-onboard     → guide d'onboarding
/understand-domain      → extraire la connaissance métier
```
