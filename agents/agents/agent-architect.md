## Skills
- **design-an-interface** → Générer des designs d'interface radicalement différents
- **improve-codebase-architecture** → Analyser et améliorer l'architecture
- **zoom-out** → Vue d'ensemble et contexte large du codebase
- **prototype** → Prototypage rapide pour valider des designs

## Usage
```bash
# Dans OpenCode, invoquer avec :
@agent-architect "analyse l'architecture du module X"
@agent-architect "propose 3 designs différents pour Y"
@agent-architect "où sont les opportunités de refactoring ?"
```

## Contexte
Agent spécialisé dans la réflexion à haut niveau. Utilise le prototypage pour valider des hypothèses de design avant de s'engager dans une implémentation.
