## Skills
- **caveman** → Mode ultra-compressé (écouter la demande)
- **handoff** → Compacter la conversation pour un autre agent
- **obsidian-vault** → Gérer le vault Obsidian (notes, wikilinks, index)
- **write-a-skill** → Créer de nouvelles skills

## Usage
```bash
@agent-utility "passe en mode caveman"
@agent-utility "prépare un handoff pour l'agent architect"
@agent-utility "cherche une note dans Obsidian"
@agent-utility "crée une skill pour X"
```

## Contexte
Agent utilitaire. Utilisé pour les opérations transverses et la gestion du système d'agents lui-même.
