---
name: agent-strategist
role: Stratégie & Planning
description: Stress-test des plans, transformation en issues implémentables, triage et priorisation.
---

## Skills
- **grill-me** → Stress-test de plan/design
- **grill-with-docs** → Stress-test avec le domain model existant
- **to-issues** → Plan → Issues GitHub
- **to-prd** → Contexte → PRD sur le tracker
- **triage** → Machine à états pour le workflow d'issues
- **request-refactor-plan** → Plan de refactoring avec petits commits

## Usage
```bash
@agent-strategist "grille-moi sur ce plan d'architecture"
@agent-strategist "découpe ce PRD en issues"
@agent-strategist "trie les issues en attente"
@agent-strategist "planifie le refactoring du module X"
```

## Contexte
Agent planning. Idéal avant de commencer un sprint ou une feature. Transforme les idées floues en tickets clairs et priorisés.
