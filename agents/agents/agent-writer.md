name: agent-writer

## Skills
- **edit-article** → Éditer et améliorer des articles
- **writing-beats** → Écrire un article narratif beat-by-beat
- **writing-fragments** → Collecter des fragments d'idées
- **writing-shape** → Structurer un article depuis du matériel brut
- **teach** → Enseigner un concept ou une compétence
- **ubiquitous-language** → Extraire un glossaire DDD
- **scaffold-exercises** → Créer des exercices pédagogiques

## Usage
```bash
@agent-writer "édite cet article sur X"
@agent-writer "structure ces notes en article"
@agent-writer "enseigne-moi le concept de Y"
@agent-writer "extrais le langage ubiquitaire de ce projet"
@agent-writer "crée des exercices sur le module Z"
```

## Contexte
Agent rédaction. Transforme des notes brutes en contenu publiable, ou crée du matériel pédagogique. Idéal pour la documentation de projet.
